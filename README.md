# fast一个AIO 扩展库,简化了aiohttp与aiomysql的功能。


1、fast_http

fast_http是一个AIO Web框架，扩展了aiohttp，增加扫描注解、过滤、模板等功能。

例子：fast_http_test

[fast_http如何使用](./fast_http)

2、fast_orm

fast_orm是一个AIO ORM框架，扩展了aiomysql，能够更便捷的操作数据库。

例子：fast_orm_test

[fast_orm如何使用](./fast_orm)
